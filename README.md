# Instagram-Clone


An instagram clone developed with react-native.

## Install



```bash
npm install
git clone https://github.com/ezgiceylan/InstagramClone.git
cd InstagramClone

```
## Usage

### Android
```bash
npx react-native start
npx react-native run-android
```
### IOS
```bash
npx react-native start
npx react-native run-ios
```
